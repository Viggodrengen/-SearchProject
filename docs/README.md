# Undervisningsmateriale

Denne mappe er ryddet op, så undervisningsmaterialet kun ligger i to formater:

- `.md` – læsbart tekstudtræk, noter, agendaer og opgavesæt.
- `.png` – billeder/sider/slides, som markdown-filerne refererer til.

PDF, PPTX, TXT, ZIP, CSV, JPG, GIF, SVG og TIFF-dubletter er fjernet fra `docs/`, fordi indholdet allerede findes som markdown + PNG-billeder.

## Start her

1. `ai_eksamensprojektet-v1-0-1.md` – eksamensprojektets krav og ramme.
2. `modul-01-agenda.md` til `modul-12-agenda.md` – agendaer for modulerne.
3. `ai_modul-*.md` – modulslides/undervisningsmateriale som markdown.
4. `ai_emne-*.md` – emnenoter som markdown.
5. `ai_opgave*.md` / `ai_opgavest*.md` – opgaver/opgavesæt som markdown.

## Filkonvention

- `modul-XX-agenda.md` – modulagenda.
- `ai_modul-*.md` – modulmateriale udtrukket fra slides/pdf.
- `ai_emne-*.md` – emnemateriale udtrukket fra pdf.
- `ai_opgave*.md` / `ai_opgavest*.md` – opgaver og guides.
- `ai_<slug>__page_###.png` – PDF-side som billede.
- `ai_<slug>__slide_###__img_##.png` – slidebillede konverteret til PNG.

## Bemærkning

De oprindelige PDF/PPTX/TXT-filer var dubletter af markdown-udtrækkene og gjorde mappen svær at navigere i. Hvis de originale kildefiler nogensinde skal bruges igen, bør de hentes fra undervisningsplatformen eller lægges i en separat arkivmappe uden for `docs/`.
